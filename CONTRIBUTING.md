# Contributing

Thanks for your interest in VoxMeet!

## Getting Started

1. Fork the repo
2. Run `npm install && npm run rebuild:native`
3. Run `npm run dev` to launch

## Pull Requests

- Keep changes focused — one feature/fix per PR
- Update tests if applicable
- Make sure `npm run build` passes

## Issues

- Search existing issues before filing a new one
- Include: macOS version, steps to reproduce, expected vs actual behavior

## Code Style

- TypeScript strict mode
- ESLint rules implicit in tsconfig
- Comments in English only
